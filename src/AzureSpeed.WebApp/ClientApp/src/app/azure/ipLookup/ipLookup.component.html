Azure IP Lookup

Azure IP Lookup tool enables you to quickly lookup service tag and region info based on ip address or domain

Restful API

Restful api is available via https://www.azurespeed.com/api/ipinfo, examples:

  • https://www.azurespeed.com/api/ipinfo?ipAddressOrUrl=104.45.231.79
  • https://www.azurespeed.com/api/ipinfo?ipAddressOrUrl=www.azurespeed.com
